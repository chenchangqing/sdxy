# DevEco Studio

## Mac下安装

1. 先注册华为账号
2. [下载zip文件](https://developer.huawei.com/consumer/cn/download/)
3. 解压`devecostudio-mac-5.0.3.404.zip`
4. 双击dmg
5. 拖拽至applications  
<img src="images/dev_eco_studio_01.png" width=100%/>
6. 不设置安装文件夹，直接点击ok  
<img src="images/dev_eco_studio_02.png" width=100%/>
7. 点击同意  
<img src="images/dev_eco_studio_03.png" width=100%/>
